
# Fine scrolling

Play with the vertical and horizontal fine scrolling registers in real-time and see the effect of switching between 40 and 38 column mode.

Keys:

  - `W`: scroll up
  - `S`: scroll down
  - `A`: scroll left
  - `D`: scroll right
  - `V`: toggle 25 or 24 line mode
  - `W`: toggle 40 or 38 column mode
